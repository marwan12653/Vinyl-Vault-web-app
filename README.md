# Vinyl-Vault-web-app

💿 Vinyl VaultA high-fidelity physical media simulation for the modern web.Vinyl Vault is an immersive, interactive record player experience built with React.js and Motion. Designed for the indie music lover who misses the tactile feel of analog, this project simulates the physical mechanics of a turntable—complete with a swinging tonearm and spinning platters—all served through a modern full-stack architecture.🛠 The Tech StackTo satisfy the Technical Requirements of the Web Engineering course, we utilized:  Frontend: React.js (Mandatory) for building a component-based UI.  Animations: Motion (Framer Motion) for hardware-accurate physics simulations.Styling: Tailwind CSS v4 for a utility-first, performant indie aesthetic.Backend: Node.js/Next.js API routes to handle RESTful data fetching.  Database: Integrated with a real database (MongoDB/PostgreSQL) to serve album metadata.  Icons: Lucide-React for clean, minimal player controls.🕹 FeaturesInteractive Crate: Browse high-res indie album art with "Vinyl Peek" hover effects.Mechanical Simulation: The tonearm (needle) physically swings onto the record when playback begins.Dynamic Turntable: Real-time rotation and CSS-generated vinyl grooves that adapt to the selected album.Hi-Fi Audio: High-quality local asset playback using the HTML5 Audio API via React useRef hooks.🏗 System ArchitectureThe application follows a standard Frontend-Backend-Database flow to ensure scalability and separation of concerns.  Client Layer: React handles the state (isPlaying, selectedAlbum) and triggers animations.API Layer: RESTful endpoints serve album metadata (Tracklists, Artist Info, Asset Paths).  Data Layer: A real database ensures no mock or static data is used for the final release.  👥 The TeamThis project was developed by a team of 2 students.
